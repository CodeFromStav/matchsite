The Problem

Motivating newcomers to contribute to Open Source Software and retaining them in the project over time stubbornly remains a perennial weakness of OSS development.

The Solution

Match Source will solve this by developing a robust mechanism for assessing and revealing the gap between a newcomer’s skills and the skills required for a particular project on its work list.
